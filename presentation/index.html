Pipeline Deep Dive

4-step AI pipeline

Step 1
Signal Denoising
Strips noise before the AI reasons over anything — schedules, commission tables, dress code, motivational filler. Keeps scripts, objections, value props, personas.
~30% noise removed avg
Step 2
Document Understanding
Tags every section: script objection-list value-prop tactical-advice and more. Training brief used as a prioritization directive.
Typed section map output
Step 3
Activity Planning
Decides which activity types to build and in what sequence. Rules: Lessons first → Memorization → RolePlays → RapidFire drills. Brief topics determine priority.
Ordered activity plan
Step 4
Content Generation
Generates a fully-typed config per activity in parallel. RolePlay uses Sonnet (complex persona + scorecard). All other types use Haiku for cost efficiency.
5 activity types · per-activity progress

Input: PDF text layer + Claude vision pass (skipped for text-dense docs) → merged canonical document

Output: ParsedTrainingConfig with ordered learningFlow[]

Security & Privacy

What leaves the machine, and how it's protected

What's sent to the LLM

Client training PDFs — containing sales scripts, objection responses, product knowledge, and customer personas.

This is the core data risk. Client data does leave the machine and must be disclosed.

Mitigations
1

Signal Denoising strips non-essential content (schedules, financial tables) before any API call — reduces data surface area

2

Anthropic API is SOC 2 Type II compliant — enterprise data handling standards, not consumer Claude

3

Locally hosted — no PDFs stored on third-party servers; Vercel Blob only used if deployed to cloud

4

HITL review catches hallucinations before anything reaches a rep — no silent automation

What's next

Two ideas to take this further

Direct API Integration

Instead of downloading a JSON file and manually importing it, the approved config is pushed directly into Chase's Replay workspace via API on export. Eliminates the last manual step entirely.

Current: Export JSON → manually import into Replay builder

Future: Click "Approve All" → activities live in Replay instantly

AI Interview Agent

Instead of a static intake form, clients receive a link to a conversational AI agent that interviews them about their sales process — tone, common objections, product nuances — then they upload their PDFs.

Current: Static form with topic checkboxes + free-text notes

Future: 10-min AI conversation → richer training brief → better pipeline output
